## Any-GPT


AnyGPT is a document-based AI assistant platform where users can upload files (PDF, DOCX, Markdown, code, etc.) and ask questions to get accurate answers with source citations. It is built on the Jaseci stack and uses a Retrieval-Augmented Generation (RAG) pipeline combining FAISS vector search, reranking, and LLM-based response generation. The current Phase 1 MVP supports document upload, semantic search, and chat-based querying with streamed responses, while being designed for future scalability into a multi-tenant, production-ready system with advanced RAG techniques, modular storage backends, and enterprise features.
